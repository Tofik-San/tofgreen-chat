# Greentoff Chat (GPT-4)

## Установка и запуск:

1. Установи зависимости:
```
npm install
```

2. В `.env` добавь:
```
OPENAI_API_KEY=твой_ключ
```

3. Запуск:
```
npm start
```

Сайт доступен на `localhost:8080`

## Особенности:
- GPT-4
- Память последних 3-х сообщений
- Адаптивный дизайн под телефон
- Диалог как в мессенджере
